Fake Image Service

以下示範是使用本服務產生的圖片：

URL 參數說明

參數說明
text要顯示的文字（預設為 {width}x{height}）
font字體名稱，對應 /public/font/ 下的 TTF 檔（不含副檔名）
retina=1開啟 Retina（2x）輸出（暫未實作，可自行擴充）
format=png強制輸出 PNG

路由結構（示範）

    /{size}/{bgColor}/{fgColor}?text=...&font=...&format=...
    /{size}.png?text=...                // 強制 PNG

更多細節請參考 GitHub Repo。
